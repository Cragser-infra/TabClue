# Phase 2: Sync & Backup

## Options
1. GitHub Gists (like NiceTab)
2. WebDAV
3. Export/Import JSON files (simplest)

## Phase 2 MVP
Start with export/import JSON:
- Export all data as a single JSON file
- Import from JSON (merge or replace)
- Import from OneTab format (text-based)

## Future
