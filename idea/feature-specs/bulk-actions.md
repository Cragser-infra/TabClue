# Bulk Actions Specification

## Selection
- Checkbox on each TabCard
- "Select All" button in ContentHeader
- Selection persists within current view/group

## Floating Action Bar
Appears when 1+ items selected. Anchored to bottom of content area.

## Actions
- Delete selected: confirmation dialog, removes from storage
- Open selected: opens all selected URLs in new tabs
- Move selected (Phase 2): move tabs to a different group

## UX
- Shows count: "X selected"
- Clear selection button
- Keyboard: Escape clears selection
